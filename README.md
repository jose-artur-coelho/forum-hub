# Desafio Forum Hub

## sobre:

Projeto de um forúm virtual desenvolvido com spring boot
desenvolvido no programa ONE parceria da Alura com a Oracle.

# Funcionalidades:

- Cadastramento de usuários;
- Autorização via JWT;
- CRUD de cursos;
- CRUD de tópicos;
- CRUD de respostas;
